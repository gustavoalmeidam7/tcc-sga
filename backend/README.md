# BACKEND PARA O TCC SGA

## Convenções:
Funções: usar Snake Case ( minha_funcao() )

Variáveis ou parâmetros privados: usar Camel Case ( meuParametro )

Constantes globais: Usar Screaming Snake Case ( MINHA_CONSTANTE )

Nome de classes e módulos: Pascal Case ( MeuModulo )

Módulos de teste: Sempre começar com os prefixos ```test_``` e ```e2e_``` ou ```unity_```, usar Camel Case ( test_e2e_Paciente )
